# GPseudoRank
Matlab code for the GPseudoRank method presented in 
ME Strauss, JE Reid, and L Wernisch. GPseudoRank: MCMC for sampling from posterior distributions of pseudo-orderings
using Gaussian processes. bioRxiv, 211417.
https://www.biorxiv.org/content/early/2017/10/31/211417
A tutorial is provided in .mlx, .m and .html format in the files GPseudRankByExample.mlx, GPseudRankByExample.m and 
GPseudRankByExample.html.
Author of the code: ME Strauss
